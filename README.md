# EnterpriseClaw

__EnterpriseClaw__ allows you to build your own AI Assistant within a corporate environment. It is designed with security principles at its core to allow it to work in heavily regulated enterprises and its based on [OpenClaw](https://openclaw.ai/)'s AI Assistant. Your assistant can receive orders from you through any corporate chat platform (initial support for Microsoft Teams and Slack) and your company's IDP (_Internal Developer Platforms_). Even though it is intended to be an individual project, it could also integrate and extend GitHub Copilot and GitHub Enterprise capabilities. This project is brought to live using a Cloud-Native approach with tools like Kubernetes, Argo, Istio, Kyverno and Kagent, to name a few.
